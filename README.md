# zhenxun_plugin_twenty_four

## 简介

适用于 zhenxun_bot 的 24 点游戏插件

~~金币回收计划项目~~


## 使用方法

1. **开启游戏**  
   发送：`开始24点`

2. **结束游戏**  
   发送：`结束24点`

3. **答题**  
   直接发送表达式，如：`(1+3)*(5+1)`  
   若无解，发送：`无解`
